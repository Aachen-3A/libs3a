Detailed Description

Database config manager class.

The database config manager is a tool to compare sample entries in a database with a local config file. It can update the contents of said file and write them to a new revision, meaning that no information is lost.

def dbconfigmanager.DBConfigManager.update_config (   self,
  sample_list,
  config_file = None 
)

Updates config_file for a given sample_list.

Searches for entries from sample_list in self.config and compares with potential database entries. If differences are found, a new config file revision with updated entires is created. Default value for config_file is the latest revision in the working directory. Return value indicates if the update was succesful.

Parameters
selfThe object pointer
sample_listList of samples to update in the config file
config_fileConfig file to update. Default is the latest revision of config files
Returns
True if the update is succesful or unnecessary, else False.
