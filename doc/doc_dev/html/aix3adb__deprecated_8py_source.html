120 class cookietransportrequest:
121  """A Transport request method that retains cookies over its lifetime.
122 
123  The regular xmlrpclib transports ignore cookies. Which causes
124  a bit of a problem when you need a cookie-based login, as with
125  the Bugzilla XMLRPC interface.
126 
127  So this is a helper for defining a Transport which looks for
128  cookies being set in responses and saves them to add to all future
129  requests.
130  """
131  # From http://www.lunch.org.uk/wiki/xmlrpccookies
132  # Inspiration drawn from
133  # http://blog.godson.in/2010/09/how-to-make-python-xmlrpclib-client.html
134  # http://www.itkovian.net/base/transport-class-for-pythons-xml-rpc-lib/
135  #
136  # Note this must be an old-style class so that __init__ handling works
137  # correctly with the old-style Transport class. If you make this class
138  # a new-style class, Transport.__init__() won't be called.
