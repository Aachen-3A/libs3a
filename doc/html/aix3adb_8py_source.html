153  """A Transport request method that retains cookies over its lifetime.
154 
155  The regular xmlrpclib transports ignore cookies. Which causes
156  a bit of a problem when you need a cookie-based login, as with
157  the Bugzilla XMLRPC interface.
158 
159  So this is a helper for defining a Transport which looks for
160  cookies being set in responses and saves them to add to all future
161  requests.
162  """
163  # From http://www.lunch.org.uk/wiki/xmlrpccookies
164  # Inspiration drawn from
165  # http://blog.godson.in/2010/09/how-to-make-python-xmlrpclib-client.html
166  # http://www.itkovian.net/base/transport-class-for-pythons-xml-rpc-lib/
167  #
168  # Note this must be an old-style class so that __init__ handling works
169  # correctly with the old-style Transport class. If you make this class
170  # a new-style class, Transport.__init__() won't be called.
